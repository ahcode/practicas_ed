ed::Donantes cargarFichero ( std::string const &  nombreFichero)

Carga un fichero de texto con una lista de donantes a un objeto Donantes.

Parámetros
nombreFichero(Nombre del fichero) de tipo string
Nota
El fichero debe tener el siguiente formato: Los donantes irán uno detrás de otro ocupando 4 líneas cada uno, 1º línea para el nombre, 2º línea para los apellidos, 3º línea para el grupo sanguíneo y 4º para el factor de riesgo
Devuelve
Donantes, objeto Donantes con los datos del fichero
Nota
Se lanzará una excepción con valor entero 1 si no se puede abrir el fichero
Se lanzará una excepción con valor entero 2 si hay un fallo al intentar añadir un grupo sanguíneo no válido
Se lanzará una excepción con valor entero 3 si hay un fallo al intentar añadir un factor de riesgo no válido
